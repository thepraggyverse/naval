# Security Policy

## Supported Versions

Security fixes are applied to the latest version on `main`.

## Reporting A Vulnerability

Please do not open a public issue for undisclosed vulnerabilities.

Report privately to:

- `praggy.dev@gmail.com`

Include:

- a clear description of the issue
- reproduction steps or proof of concept
- impact assessment
- suggested mitigation, if you have one

## Scope Notes

This repository primarily contains skill instructions, reference markdown, JSON manifests, and local install scripts.

- It does not run a server.
- It does not include telemetry.
- It does not include the full book text.
- Security and privacy behavior also depend on the host AI tool and model provider you use.

For data-handling details, see [PRIVACY.md](PRIVACY.md).
