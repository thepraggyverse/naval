# Privacy & Data Handling

`naval` is a markdown/config skill pack. It does not include telemetry, analytics, a background service, or network-calling code.

## What The Repository Contains

- `SKILL.md` skill instructions
- source-safe reference files
- install and validation scripts
- plugin manifests for supported harnesses

## What May Send Data

Your AI host or model provider may send prompts, files, or workspace context to its configured model service. That behavior is controlled by the host tool and provider, not by this repository.

The included Python scripts validate files, update local marketplace metadata, or create local symlinks. They do not upload project data.

## Book Text

This repository does not include the full text of *The Almanack of Naval Ravikant*. It contains paraphrased skill instructions, workflow templates, and coverage metadata.

## Reporting

For security or privacy concerns, follow [SECURITY.md](SECURITY.md).
