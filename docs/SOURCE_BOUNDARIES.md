# Source Boundaries

This repository is built from a local reading of *The Almanack of Naval Ravikant* by Eric Jorgenson.

The public repo intentionally contains:

- paraphrased principles
- operational workflows
- skill routing
- coverage maps
- reading and review scaffolding

The public repo intentionally does not contain:

- the full book text
- long excerpts
- the source PDF
- copied chapter passages

## Citation And Quote Use

Use `n-source-fidelity` or `n-quote-safety` when producing quotation-like output.

Default behavior:

- Paraphrase.
- Keep direct excerpts short.
- Mention page or section when available.
- Verify exact phrasing against the original book or a primary Naval source before formal citation.
