# `n-presence-practice`

**Area:** happiness

**Use when:** Reduce rumination and anticipation by returning attention to present reality.

**Example prompt:**

```text

```

## What It Does

Apply The Almanack of Naval Ravikant to reduce rumination and anticipation by returning attention to present reality. Use when the user asks for n-presence-practice, says "Help me return to the present.", or wants this Naval lens.

## How To Use

Ask the agent to use `$n-presence-practice` or say `n-presence-practice` in the prompt. If you are unsure which skill fits, start with `n-router`.
