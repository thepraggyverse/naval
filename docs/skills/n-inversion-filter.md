# `n-inversion-filter`

**Area:** judgment

**Use when:** Find what will fail, what to avoid, and which mistakes matter most.

**Example prompt:**

```text
How could this fail?
```

## What It Does

Apply The Almanack of Naval Ravikant to find what will fail, what to avoid, and which mistakes matter most. Use when the user asks for n-inversion-filter, says "How could this fail?", or wants this Naval lens.

## How To Use

Ask the agent to use `$n-inversion-filter` or say `n-inversion-filter` in the prompt. If you are unsure which skill fits, start with `n-router`.
