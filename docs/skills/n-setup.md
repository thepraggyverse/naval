# `n-setup`

**Area:** meta

**Use when:** Configure optional Naval memory output paths, privacy defaults, and direct-install reference guidance.

**Example prompt:**

```text
Set up Naval memory for this repo.
```

## What It Does

Configure optional Naval memory output paths, privacy defaults, and direct-install reference guidance. Use when the user asks to set up Naval memory, choose where reviews/learnings should be saved, configure .naval/config.local.yaml, or prepare copied/direct skill installs.

## How To Use

Ask the agent to use `$n-setup` or say `n-setup` in the prompt. If you are unsure which skill fits, start with `n-router`.
