# `n-next-sources`

**Area:** reading

**Use when:** Point users to deeper Naval resources, podcasts, essays, and source trails.

**Example prompt:**

```text
Where should I go deeper on this topic?
```

## What It Does

Naval next sources: Point users to deeper Naval resources, podcasts, essays, and source trails. Trigger when the user asks "Where should I go deeper on this topic?" or names n-next-sources.

## How To Use

Ask the agent to use `$n-next-sources` or say `n-next-sources` in the prompt. If you are unsure which skill fits, start with `n-router`.
