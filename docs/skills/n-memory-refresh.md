# `n-memory-refresh`

**Area:** meta

**Use when:** Refresh saved Naval memory entries for stale, contradictory, duplicate, or low-value guidance.

**Example prompt:**

```text
Refresh my saved Naval learnings.
```

## What It Does

Refresh saved Naval memory entries for stale, contradictory, duplicate, superseded, or low-value guidance. Use when the user asks to audit docs/naval, clean saved learnings, refresh Naval memory, or resolve contradictions in saved reviews and scorecards.

## How To Use

Ask the agent to use `$n-memory-refresh` or say `n-memory-refresh` in the prompt. If you are unsure which skill fits, start with `n-router`.
